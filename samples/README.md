# samples

Sample inputs to drive the dev flow without preparing your own. See the
project root `README.md` for the click-through.

- `chat_transcript.json` — three user turns; paste each into the chat panel
- `context.md` — one-page fabricated product brief; upload as a context file
- `goal.txt` — the goal string you'd type when starting the simulation
- `screenshots/01..04*.png` — drag into the Shots tab on the left panel
